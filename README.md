# Frostmoln Scoop Bucket

Scoop bucket for the [Frostmoln](https://frostmoln.cloud) CLI, `fm`, on Windows.

## Install

```powershell
scoop bucket add frostmoln https://github.com/Frostmoln/scoop-bucket
scoop install fm
```

## Notes

- Manifests are generated automatically by goreleaser on each `fm` release. Do not edit by hand.
- Binaries are hosted in the EU on `get.frostmoln.cloud`; this repo holds only the manifest.
